# 🚀 HƯỚNG DẪN CHẠY ĐỒ ÁN TRỰC TUYẾN TRÊN TRÌNH DUYỆT

Để chạy chương trình thi trắc nghiệm này ngay lập tức mà không cần cài đặt phần mềm, bạn làm theo 3 bước sau:

1. Nhìn lên góc phải màn hình GitHub này, bấm vào nút **Code** (Màu xanh dương) $\rightarrow$ Chọn tab **Codespaces** $\rightarrow$ Bấm nút **Create codespace on main**.
2. Đợi khoảng 10-15 giây để GitHub thiết lập máy ảo Linux. Một giao diện VS Code online sẽ hiện ra ngay trên trình duyệt.
3. Nhìn xuống ô **Terminal** ở dưới cùng màn hình mạng, copy lệnh sau dán vào rồi ấn **Enter** là chương trình khởi động luôn:
   ```bash
   g++ bai5.cpp -o bai5 && ./bai5
